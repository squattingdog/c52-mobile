# Mobile app
This is the mobile app project.  It is an Ionic 4 with Angular 6 app.

## Dev Setup

### Install Cordova

``` cli
npm install -g cordova
```

### Install Ionic

``` cli
npm install -g ionic
```

### Install dependencies

``` cli
npm install
```

## Run the app

### Web / Browser
``` cli
ionic serve
```

### Android physical device or AVD (started)

``` cli
ionic cordova run android
```

### IOS

``` cli
ionic cordova run ios
```
